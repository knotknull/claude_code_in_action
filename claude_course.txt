# Github Integration 

/install-github-app: sets up claud github actions for a repo

# default github actions 
Mention Action 
 - Claude mentions with "@claude" 
 - Claude Code runs
 - Claude completes the task it was given with the mention
 - Claude Code responds 
      - response added to the issue or PR

Pull request actions
 - create a pull request
      (triggers an action)
 - Claude Code runs
 - Claude Code reviews changes
 - Claude Code posts a full report on the impact of the pull request

 NOTE: In github, create an issue and in description include 
 @claude can you take a look at this, Just test them out and verify 
 that clicking them toggles between code and preview tabs
